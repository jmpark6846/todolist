# todolist
Wintercoding 과제: todo list✍ 만들기  
https://wintercoding-todolist.herokuapp.com/

## 설치 순서

    git clone <url> 
    cd onelist
    virtualenv todolist
    source todolist/bin/activate
    pip install -r requirements.txt
    python manage.py migrate
    python manage.py runserver
    
## 개발/배포 환경
- Python 3.6.6
- Django 2.12
- sqlite3
- Ubuntu 18.04
- jQuery 3.3.1
- jQuery UI 1.12.1
- Bootstrap 4.1.3 
- heroku
